# 📊 Exploratory Data Analysis (EDA) – Superstore Sales Dataset

## 📌 Overview
This project focuses on performing **Exploratory Data Analysis (EDA)** on the Superstore Sales dataset. The dataset is widely used for educational purposes and contains a rich mix of **categorical, numerical, and date-time features**, making it ideal for practicing data cleaning, transformation, and visualization techniques.

The primary objective is to systematically explore the dataset, identify patterns and trends, and prepare the data for further analysis or predictive modeling.

---

## 🎯 Objectives
- Perform initial data exploration and understand dataset structure  
- Identify and handle missing values and duplicate records  
- Standardize and clean date fields  
- Resolve inconsistencies in key attributes  
- Perform feature engineering for better insights  
- Prepare clean and structured data for further analysis  

---

## 🛠️ Tools Used
- Python 🐍  
- Pandas  
- NumPy  
- Matplotlib  
- Exploratory Data Analysis (EDA) techniques  

---

## 📁 Dataset Information
- **Total Columns:** 21  
- **Type:** Retail Sales Data  
- **Domain:** Business Analytics / E-commerce  
- **Use Case:** Data Analysis, Visualization, and Feature Engineering  

---

## 🧾 Key Features
- Row-level transaction data with unique identifiers  
- Order and shipping details with date-time attributes  
- Customer segmentation information  
- Product hierarchy (Category → Sub-Category → Product Name)  
- Sales, discount, quantity, and profit metrics  
- Geographic information (Country, City, State, Region)  

---

## 🧹 Data Cleaning & Preprocessing Steps
- Removed duplicate records  
- Handled missing values in key columns  
- Standardized date formats for consistency  
- Extracted and validated year from Order ID and Order Date  
- Created new features such as **Days to Ship**  
- Corrected inconsistencies across date-related fields  

---

- Data validation and consistency checks  
- Descriptive statistical analysis

The dataset is transformed into a clean and analysis-ready format, enabling deeper insights into sales performance, customer behavior, and operational efficiency.
